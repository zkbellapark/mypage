Claude Code

터미널에서 AI로 웹사이트 만드는 법

AI 코딩 어시스턴트 입문 가이드 · 코딩 경험 제로에서 시작

Section 01

Claude Code란?

Claude Code는 터미널(명령 프롬프트)에서 실행하는 AI 코딩 어시스턴트입니다. 자연어로 지시하면 코드 작성, 수정, 디버깅, 파일 관리까지 알아서 처리합니다.

웹 채팅과 달리 내 프로젝트 폴더 전체를 이해하고, 직접 파일을 만들고 편집하고 터미널 명령까지 실행합니다.

코딩판 ChatGPT인데, 내 컴퓨터에서 직접 코드를 짜주는 버전

왜 ChatGPT가 아니라 Claude Code인가?

01

프로젝트 전체를 이해합니다

ChatGPT는 코드를 복붙해서 보여줄 뿐이지만, Claude Code는 프로젝트 폴더 안의 모든 파일을 읽고 맥락을 파악한 뒤 직접 수정합니다.

02

파일을 직접 생성하고 편집합니다

"랜딩페이지 만들어줘"라고 하면 ChatGPT는 코드 텍스트를 보여주지만, Claude Code는 실제 index.html 파일을 만들어서 저장합니다.

03

터미널 명령을 직접 실행합니다

npm install, git commit 같은 명령도 Claude가 알아서 실행해줍니다. 복사-붙여넣기 없이 자연어 한 줄이면 끝입니다.

04

200K 토큰 컨텍스트 윈도우

긴 코드베이스도 한 번에 이해합니다. 파일 수십 개짜리 프로젝트도 전체 맥락을 놓치지 않습니다.

05

에이전틱(Agentic) 코딩

단순 질답이 아니라, 스스로 계획을 세우고 → 코드를 짜고 → 테스트하고 → 에러를 고치는 자율적 워크플로우를 수행합니다.

ChatGPT/Copilot = 코드 제안 도구 │ Claude Code = 코드를 직접 실행하는 AI 개발자

Section 03

Claude Code vs Cowork vs Chat

Anthropic에는 3가지 방식으로 Claude를 쓸 수 있습니다. 각각 용도가 다릅니다.

💬

Claude Chat

웹/앱 (claude.ai)

대화, 질문, 글쓰기

⌨️

Claude Code

터미널 (CLI)

코딩, 파일 생성/편집

🖥️

Cowork

데스크톱 앱

파일 관리, 업무 자동화

구분Claude ChatClaude CodeCowork
파일 접근업로드한 것만프로젝트 폴더 전체내 컴퓨터 파일 전체
코드 실행제한적직접 실행직접 실행
주 사용자일반 사용자개발자비개발자 업무 자동화
난이도쉬움터미널 필요쉬움
코딩으로 뭔가 만들고 싶다면 (웹사이트, 앱, 자동화 등) → Claude Code
문서 정리, 이메일 자동화 등 비코딩 업무 → Cowork
그냥 대화하면서 질문하기 → Chat
Section 04

랜딩 페이지 만들어보기

코딩 경험이 전혀 없어도 괜찮습니다. 아래 순서대로 따라해보세요.

STEP 0

터미널 열기

Mac: Spotlight (Cmd+Space) → "Terminal" 검색 → 엔터

Windows: 시작 메뉴 → "PowerShell" 검색 → 엔터

STEP 1

Node.js 설치 (한 번만)

nodejs.org 접속 → LTS 버전 다운로드 → 설치

node --version   # v20.x.x 같은 숫자가 나오면 성공!
STEP 2

Claude Code 설치 (한 번만)

npm install -g @anthropic-ai/claude-code

설치 후 아무 곳에서나 claude 명령어를 쓸 수 있게 됩니다.

STEP 3

프로젝트 폴더 만들기

mkdir my-landing-page
cd my-landing-page

mkdir = 폴더 만들기, cd = 그 폴더로 이동하기

STEP 4

Claude Code 실행!

$ claude

첫 실행 시 Anthropic 계정 로그인 화면이 뜹니다. 로그인하면 준비 완료!

STEP 5

랜딩페이지 만들기

Claude Code에 이렇게 입력하세요:

프롬프트 예시
"심플한 원페이지 랜딩페이지를 만들어줘.
히어로 섹션에 제목과 부제목,
서비스 특징 3개 카드,
가격표 섹션,
하단에 CTA 버튼 넣어줘.
한국어로 작성하고 모바일 반응형으로."
STEP 6

결과 확인

Claude가 파일을 만든 뒤, 브라우저에서 바로 확인:

open index.html   # Mac
start index.html  # Windows

💡 Tip

  • 구체적으로 지시할수록 결과가 좋습니다 (색상, 폰트, 레이아웃 등)
  • 마음에 안 들면 "히어로 배경을 그라데이션으로 바꿔줘" 처럼 수정 요청
  • /compact 명령으로 토큰을 절약하세요
  • /cost 명령으로 사용량을 확인할 수 있습니다
